### 🧭 &nbsp; Philosophy

> **GOTCHA before vibe.** &nbsp; Goals · Orchestration · Tools · Context · Hard prompts · Args.
> Six layers. Composable, debuggable, boring in the right places so the agent can be interesting in the useful ones.

Agents should remember. Memory should be tiered. Prompts should be cached. Tokens should be compressed. Everything else is a leaky abstraction.
